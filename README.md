# Exchange

### Usage

    make venv
    make currencies - добавление стандартных валют(опционально)
    make up



Сваггер доступен по /api

### Add default currencies:
    make currencies    
### UP
    make up

### Clear db
    make db_clear
    
### Create venv:
    make venv

### Run tests:
    make test
    
### Run linters:
    make lint
    
### Run formatters:
    make format
